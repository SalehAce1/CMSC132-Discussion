Command Line Execution¶

We are going to now take a look at how to program in Java without using Eclipse. Instead of Eclipse, we are going to use our command line.

To do this, download the file and then open up command line (Windows) or terminal (MacOS) or shell (Linux).

Basic Commands¶

  • javac is the command that compiles a .java file. This will create a .class file.

    • Format of the command : javac FILENAME.java

  • java is the command that runs the program.

    • Format of the command : java FILENAME args

Note

The FILENAME parameter does not include the file extension. Additionally, the args parameter is a list of arguments. (For CommandLine.java , args is a set of integers) Example: java CommandLine 1 2 3 4

Note

args represents the String[] args in the main method header public static void main(String[] args)

  • nano is the command that opens a very basic editor to write code in your terminal.

    • Format of the command : nano FILENAME.java
